# 需求分析文档（BRD/MRD）
## 项目：PixSell AI — 跨境卖家 AI 营销素材生成平台
**版本：** v1.0  |  **日期：** 2026-05-16  |  **角色：需求分析师**

---

## 一、项目背景

基于市场调研（详见 01-market-research.md），200万+ 中国跨境卖家面临视觉素材生产效率低、成本高的核心痛点。现有工具均为英文产品，未能服务这一群体。PixSell AI 以"中文界面 + 专业合规 + 多平台支持"为差异化，切入跨境卖家 AI 图片生成赛道。

## 三、核心业务需求（Business Requirements）

### BR-01：一键白底主图生成
**来源：** Persona A/B/C 共同需求  
**描述：** 用户上传1张原始产品照片（任意背景、任意角度），系统自动生成符合亚马逊合规标准的纯白底主图。  
**验收标准：**
- 背景 RGB = 255,255,255（误差±2）
- 产品占图面积 ≥ 85%
- 图片分辨率 ≥ 2000×2000px
- 产品边缘自然，无明显锯齿
- 处理时间 ≤ 30秒

### BR-02：场景图/生活方式图生成
**来源：** Persona A/B  
**描述：** 用户选择场景模板或输入中文描述，系统生成产品在生活场景中的展示图。  
**验收标准：**
- 提供 ≥ 50 种预设场景模板（按类目分类）
- 支持中文自定义描述（系统内部翻译为英文 Prompt）
- 输出分辨率 ≥ 1600×1600px
- 产品形态保真度高（不变形、不幻觉）

### BR-03：多平台规格适配
**来源：** Persona A/B  
**描述：** 同一张图可一键导出多平台所需规格。  
**验收标准：**

| 平台 | 规格 |
|------|------|
| Amazon 主图 | 2000×2000px JPEG/PNG |
| Amazon A+ 内容 | 970×600px，970×300px，300×300px |
| TikTok Shop 主图 | 800×800px |
| TikTok 短视频封面 | 1080×1920px（9:16） |
| Shopify 商品图 | 2048×2048px |
| Temu 主图 | 800×800px 白底 |

### BR-04：A+ 内容图生成（亚马逊增强型品牌内容）
**来源：** Persona B/C  
**描述：** 生成亚马逊 A+ 页面所需的模块化图片（横幅、特性展示、比较图）。  
**验收标准：**
- 支持 ≥ 5 种 A+ 模块类型
- 可添加英文文字叠加（非主图）
- 保持品牌配色一致

### BR-05：批量处理
**来源：** Persona B/C  
**描述：** 支持一次上传多张产品图，批量生成所有类型素材。  
**验收标准：**
- 支持 CSV 导入（包含产品名称、类目、描述）
- 单次批处理 ≥ 50 SKU
- 提供进度条和队列管理
- 完成后可一键打包下载（ZIP）

### BR-06：亚马逊合规自动检测
**来源：** Persona A/B/C  
**描述：** 对生成的主图进行自动合规检测，标注不通过项并给出修复建议。  
**验收标准：**
- 检测项：背景颜色（RGB白度）、产品占比、文字叠加检测、水印检测、图片尺寸
- 输出：通过/警告/不通过 + 具体问题描述
- 可对不通过的图片一键重新生成

### BR-07：用户账户与工作空间管理
**来源：** Persona B/C  
**描述：** 支持多品牌/多工作空间管理，团队成员协作。  
**验收标准：**
- 每个工作空间独立存储素材
- 支持邀请团队成员（按 Pro/Business 套餐限制席位数）
- 品牌风格保存（Logo、主色调、偏好场景风格）

### BR-08：订阅与支付
**来源：** 商业需求  
**描述：** 分层订阅定价，支持 Stripe 支付。  
**验收标准：**
- 提供免费体验额度
- 支持月付/年付（年付享 20% 折扣）
- Credits 不过期（参考 Nozam 模式）
- 支持升级/降级/取消订阅

## 四、非功能需求（Non-functional Requirements）

### NFR-01：性能
- 单张图片生成（白底主图）响应时间 ≤ 30秒
- 场景图生成 ≤ 60秒
- 批量处理：50 SKU 全部完成 ≤ 30分钟
- 页面加载时间 ≤ 2秒（首屏）

### NFR-02：可用性
- 界面语言：**中文（简体）为主**，英文为辅
- 支持中文 Prompt 输入（产品描述、场景描述）
- 所有错误提示为中文
- 移动端响应式支持（使用手机查看结果）

### NFR-03：可靠性
- 系统可用性 ≥ 99.5%
- 图片生成失败时自动重试 ≤ 3次
- 失败的任务不消耗 credits
- 数据备份：用户生成的图片至少保留 **90天**

### NFR-04：安全性
- HTTPS 全站加密
- 图片存储 URL 带签名过期（防盗链）
- 用户产品照片不用于训练第三方模型（需在 ToS 中明确声明）
- 符合中国《数据安全法》和欧盟 GDPR 基本要求

### NFR-05：可扩展性
- API 接口预留（Business 套餐可开启）
- 支持接入新的 AI 图像生成模型（模块化替换）
- 支持新增平台规格（插件化配置）

## 五、系统边界与约束

### 5.1 MVP 范围（v1.0，第1–2月）
✅ 白底主图生成  
✅ 3种场景模板生成  
✅ 亚马逊规格导出  
✅ 基础合规检测  
✅ 用户注册/登录  
✅ Stripe 订阅  
✅ 图片历史管理  

### 5.2 MVP 不包含
❌ A+ 内容生成  
❌ 视频生成  
❌ 批量 CSV 导入  
❌ 多工作空间  
❌ API 接口  
❌ TikTok Shop 独立规格支持  

### 5.3 技术约束
- AI 图像生成：使用第三方 API（Replicate / OpenAI）而非自研模型
- 图片存储：Cloudflare R2 或 AWS S3
- 部署：Vercel（前端）+ PostgreSQL on Railway/Supabase
