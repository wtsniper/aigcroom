# 产品需求文档（PRD）
## 项目：PixSell AI — 跨境卖家 AI 营销素材生成平台
**版本：** v1.0  |  **日期：** 2026-05-16  |  **角色：产品经理**

---

## 一、产品愿景

**"让每一个中国跨境卖家，都能用 AI 做出专业级的英文电商素材。"**

PixSell AI 是专为中国出海卖家设计的 AI 营销素材生成平台。无需设计技能，上传一张产品照片，30 秒内获得符合亚马逊/TikTok Shop/Shopify 规格的专业素材，成本仅为外包的 1/100。

---

## 二、产品核心价值主张

1. **中文界面，零门槛上手** — 用中文描述，系统自动转换为 AI 模型需要的英文 Prompt
2. **一键合规** — 自动检测亚马逊白底要求（RGB 255）、产品占比、禁用元素
3. **多平台一键导出** — 一张图，自动生成 Amazon/TikTok Shop/Shopify 所有规格
4. **专为跨境场景训练的模板** — 针对家居、服装、电子、宠物等主流类目优化

## 三、功能规格详解（Feature Specifications）

### F-001：用户认证与账户系统

**优先级：** P0（MVP 必须）

**功能描述：**
- 邮箱注册/登录
- Google 第三方登录（方便海外华人）
- 微信登录（国内用户）
- 密码找回
- 邮箱验证

**用户流程：**
1. 访问首页 → 点击"免费开始"
2. 输入邮箱 + 密码（或选择 Google/微信登录）
3. 验证邮箱
4. 进入新手引导（Onboarding）
5. 自动赠送 10 个免费 credits

**页面：** 注册页、登录页、找回密码页、邮箱验证页

---

### F-002：新手引导（Onboarding）

**优先级：** P0（MVP 必须）

**功能描述：**
用户首次登录后，通过 3 步引导快速完成第一张图片生成，感受核心价值。

**流程：**
1. 步骤1：选择主要销售平台（Amazon / TikTok Shop / Shopify / 多平台）
2. 步骤2：选择主要销售类目（家居、服装、电子3C、宠物、运动户外、其他）
3. 步骤3：立即上传第一张产品图，生成白底主图

---

### F-003：图片生成 — 白底主图（Amazon Main Image）

**优先级：** P0（MVP 必须）

**功能描述：**
一键将任意背景的产品照片转换为亚马逊合规的纯白背景主图。

**详细功能点：**

| 功能点 | 描述 |
|--------|------|
| 图片上传 | 支持拖拽/点击上传，格式：JPG/PNG/WEBP，最大 20MB |
| 背景移除 | AI 自动识别并移除背景，保留产品 |
| 白底合成 | 背景替换为 RGB(255,255,255) 纯白 |
| 产品居中 | 自动调整产品位置使其居中且占比 ≥ 85% |
| 阴影添加 | 可选：添加自然地面阴影，增加立体感 |
| 边缘优化 | 产品边缘抗锯齿处理，毛发/透明物体特殊处理 |
| 输出尺寸 | 2000×2000px（标准），可选 1000×1000px |
| 输出格式 | JPEG（白底主图）/ PNG（透明底，用于 Photoshop 二次编辑） |

**交互设计：**
- Before/After 对比滑块预览
- 可调整产品大小（±20% 微调）
- 可调整产品位置（上下左右微调）
- 一键重新生成（消耗同等 credits）
- 满意后点击"下载"或"保存到工作空间"

**合规检测（实时）：**
- 上传后实时显示检测结果
- 绿色 ✅：通过 | 黄色 ⚠：警告 | 红色 ❌：不通过
- 检测项：背景颜色、产品占比、文字/水印检测

---

### F-004：图片生成 — 场景/生活方式图

**优先级：** P0（MVP 必须）

**功能描述：**
将产品图与生活场景合成，生成展示产品使用场景的图片。

**详细功能点：**

**场景模板库（按类目分组）：**

| 类目 | 预设场景（各5–8个） |
|------|-----------------|
| 家居 | 客厅沙发旁、厨房台面、卧室床头、户外庭院、阳台... |
| 服装/配饰 | 城市街头、咖啡馆、户外公园、健身房... |
| 电子3C | 办公桌、咖啡馆工作场景、户外使用... |
| 宠物用品 | 家庭地板、宠物角、户外草地... |
| 运动户外 | 山地、海边、健身房、公园... |
| 通用 | 渐变色背景（多色）、木质纹背景、大理石背景... |

**中文场景描述输入：**
- 用户可输入中文描述："白色现代简约的客厅，阳光明媚，有绿植装饰"
- 系统自动翻译并构建 AI Prompt
- 显示预估生成效果标签（"温暖家居感"、"高级感"、"生活化"）

**产品一致性保证：**
- 使用 IP-Adapter 或 ControlNet 技术保证产品形态不变形
- 实时预览模式（低分辨率快速预览，满意后生成高分辨率）

---

### F-005：A+ 内容图生成

**优先级：** P1（v1.1 添加）

**功能描述：**
生成亚马逊 A+ 品牌内容所需的模块化图片。

**支持模块类型：**
1. 横幅Banner（970×600px）— 产品 + 品牌名 + Slogan
2. 特性展示图（300×300px + 文字）— 展示产品某一特性
3. 比较表格图 — 本品 vs 竞品（可视化对比）
4. 生活方式大图（970×300px）
5. 产品细节特写图

**文字内容：**
- 用户输入中文卖点 → 系统自动翻译为专业英文广告语
- 支持手动修改英文文案
- 字体/颜色从品牌设置中自动继承

---

### F-006：多平台规格导出

**优先级：** P0（MVP 必须）

**功能描述：**
生成结果页提供一键多规格导出，无需用户手动裁剪。

**导出规格配置：**

```
[Amazon 主图]     2000×2000 JPEG
[Amazon 附图]     2000×2000 JPEG  
[Amazon A+ 横版]  970×600  JPEG
[TikTok Shop]    800×800  JPEG
[Shopify 商品图]  2048×2048 PNG
[Temu 白底图]     800×800  JPEG
[Instagram 方图]  1080×1080 JPEG
[自定义规格]      用户输入宽×高
```

**下载方式：**
- 单张下载
- 全规格打包下载（ZIP）
- 保存到"图片库"

---

### F-007：批量处理

**优先级：** P1（v1.1，Pro/Business 套餐）

**功能描述：**
一次上传多张产品图，批量生成所有素材。

**操作流程：**
1. 上传方式1：批量上传图片文件（最多 50 张/次）
2. 上传方式2：CSV 导入（含图片 URL + 产品名 + 类目 + 描述）
3. 选择生成类型：白底主图 / 场景图（选一种场景模板） / 全部
4. 系统后台队列处理，实时显示进度（已完成X/总计X）
5. 完成后通知（站内 + 邮件），一键下载 ZIP

**CSV 格式模板：**
```csv
image_url,product_name,category,description,scene_template
https://...,无线蓝牙耳机,电子3C,ANC主动降噪,办公桌场景
```

---

### F-008：合规检测工具

**优先级：** P0（MVP 必须，免费提供）

**功能描述：**
独立工具页面（也集成在生成流程中），检测任意上传图片是否符合亚马逊规范。

**检测项目详单：**

| 检测项 | 标准 | 判定 |
|-------|------|------|
| 背景颜色 | 纯白 RGB(255,255,255)，误差 ≤ 5 | ✅/⚠/❌ |
| 产品占比 | ≥ 85% | ✅/⚠/❌ |
| 图片分辨率 | ≥ 1000×1000px（推荐≥2000） | ✅/⚠ |
| 文字叠加 | 无文字/Logo 叠加 | ✅/❌ |
| 水印检测 | 无水印 | ✅/❌ |
| 边框检测 | 无边框、无颜色渐变 | ✅/❌ |
| 多产品检测 | 仅含单一商品（变体规则例外） | ✅/⚠ |

**输出报告：**
- 总体：通过/存在问题
- 每项详细说明（中文）
- 修复建议（"建议增大产品尺寸至占图85%以上"）
- 一键用 PixSell AI 修复按钮

---

### F-009：图片库（Asset Manager）

**优先级：** P0（MVP 必须）

**功能描述：**
管理用户所有历史生成的图片。

**功能点：**
- 按时间/产品名称/类目排序筛选
- 图片缩略图网格视图
- 点击查看原图和所有导出版本
- 删除（释放存储空间）
- 再次导出不消耗 credits
- 支持批量删除

**存储策略：**
- 免费用户：最近 30 天内的图片，最多 50 张
- Starter：90 天，最多 200 张
- Pro/Business：无限期，无数量限制

### F-011：订阅与支付

**优先级：** P0（MVP 必须）

**功能描述：**
Stripe 驱动的订阅系统。

**功能点：**

## 五、核心用户流程（User Flows）

### Flow 1：新用户生成第一张白底主图

```
进入首页 
→ 点击「免费开始」 
→ 注册/Google登录 
→ 新手引导（选平台+类目）
→ 进入生成页 
→ 上传产品图
→ 系统自动生成白底主图（进度条）
→ 预览结果（Before/After 对比）
→ 合规检测结果实时展示
→ 点击「下载」或「保存到图片库」
→ 弹出升级提示（本月还剩 8 credits，升级获得更多）
```

### Flow 2：老用户批量处理

```
登录 → Dashboard 
→ 点击「批量处理」
→ 选择上传方式（批量图片 or CSV）
→ 上传文件（最多50张）
→ 选择生成类型 + 场景模板
→ 确认 Credits 消耗（系统预估：X credits）
→ 提交任务 → 后台队列处理
→ 收到邮件/站内通知「处理完成」
→ 进入「图片库」查看 / 一键下载 ZIP
```

### Flow 3：合规检测

```
进入「合规检测」页面
→ 上传任意产品图（可拖拽多张）
→ 实时显示检测进度（逐项检测动画）
→ 展示检测报告（每项 ✅/⚠/❌ + 说明）
→ 不通过项 → 「一键用 PixSell AI 修复」按钮
→ 点击修复 → 跳转到白底主图生成，原图自动导入
```

## 六、关键界面设计规范

### 设计风格
- 风格：现代、极简、专业、深色/浅色模式双支持
- 主色调：深蓝 (#1E40AF) + 亮橙 (#F97316)（体现「跨境」活力感）
- 字体：中文「思源黑体」，英文「Inter」
- 整体感：高级感 > 廉价感，参考 Notion / Linear 的 UI 风格

### 核心组件
- 大号 Before/After 图片对比滑块（生成结果展示核心组件）
- Credits 余额徽章（全局 Header 显示）
- 任务进度条（生成中状态）
- 合规状态徽章（✅⚠❌）
